== Description ==

Shows social media buttons under posts. Note that this plugin does not track
your site or users but the share buttons of the services shown may behave
differently. Please refer to related service privacy and terms documents.


== Installation ==

1. Download the last version of the plugin from its page.
2. Upload the .zip through the Plugins/Add new/Upload of the WordPress administrator.
3. Activate the plugin through the Plugins menu of the WordPress administrator.
