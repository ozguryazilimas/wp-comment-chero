== Description ==

This plugin activates Wordpress oEmbed capabilities for comments.


== Installation ==

For WordPress 3.+:

1. Download the latest version of the plugin from its page.
1. Upload the .zip through the Plugins/Add new/Upload of the WordPress administrator.
1. Activate the plugin through the Plugins menu of the WordPress administrator.

This plug in will work as is.
