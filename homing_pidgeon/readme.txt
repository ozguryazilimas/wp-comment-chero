== Description ==

Notifies configured users when a post is submitted as pending


== Installation ==

For WordPress 4.+:

1. Download the last version of the plugin from its page.
1. Upload the .zip through the Plugins/Add new/Upload of the WordPress administrator.
1. Activate the plugin through the Plugins menu of the WordPress administrator.
1. Pull the widget named Fluxophy into your sidebar.


= What are the options for this plug in? =

1. Title: This is what appears on the widget.  Default: Fluxophy
1. Source URL: Address where the data will be fetched from
1. Display Count: Number of entries to show
