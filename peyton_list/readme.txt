== Description ==

This plugin lists whatever you like with ability to make a list with catgories
and links. This is dedicated to the lovely Peyton List (the more beautiful
one, with black hair).


== Installation ==

For WordPress 3.+:

1. Download the latest version of the plugin from its page.
1. Upload the .zip through the Plugins/Add new/Upload of the WordPress administrator.
1. Activate the plugin through the Plugins menu of the WordPress administrator.

This plug in will work as is. Just add '[peyton_list]' shorttag to a post or page in your Wordpress site.
